# Smart Warehouse AI Monitoring System

## Overview

An AI-powered Warehouse Management System that continuously analyzes inventory data to detect risks, predict shortages, identify overstock, and monitor expiring items.

---

## Features

* Automated inventory monitoring
* Expiry detection
* Demand prediction
* Overstock alerts
* Anomaly detection
* Action recommendations

---

## Architecture

Frontend → Backend → Database → AI Engine → Alerts Dashboard

---

## Tech Stack

Frontend: React
Backend: Flask / Node.js
Database: MySQL / PostgreSQL
AI: Python + Scikit-learn

```

Run backend:

```

```

Run AI monitor:

```
python ai_engine/monitor.py
